# Modified Printrun

This is a modified version of [Printrun](https://github.com/kliment/Printrun), an open-source software project distributed under the GNU General Public License v3. 

## Modifications
- text-to-gcode.py:
  * The program was modified to accept command line inputs as a list of arguments rather than a separate text file.
  * Addition of scalar function to scale the final printed text by any value up or down.
  * Addition of code to change federate of the CNC machine from command line
  * Addition of arguments which allow the program to dictate where the plotting will begin with starting x and y offsets
